<p>Fazendo parte do propósito da prática de programação, esse projeto amplia as funcionalidades do projeto DevClub Convert Money,
que tinha como objetivo criar um conversor de moeda, convertendo da moeda brasileira (Real) para o (Dolar) e o (Euro).
No desafio ampliou-se para outras duas moedas a Libra Esterlina e o BitCoin. O projeto foi composto pelas linguagens HTML, CSS e JAVA SCRIPT.</p>
